# Output Folder Structure

All generated files are saved under the `outputs/` directory, organized by type and module.

```
outputs/
├── data/                              # All CSV datasets (shared across modules)
│   ├── student_learning_behavior.csv  # Generated by: data_generator.py
│   ├── preprocessed_train.csv         # Generated by: module2_preprocessing.py
│   ├── preprocessed_test.csv          # Generated by: module2_preprocessing.py
│   └── preprocessed_full.csv          # Generated by: module2_preprocessing.py
│
├── plots/                             # All visualizations
│   ├── module2_preprocessing/
│   │   └── correlation_heatmap.png
│   ├── module4_classification/
│   │   ├── classification_roc_curves.png
│   │   ├── classification_pr_curves.png
│   │   ├── classification_confusion_matrices.png
│   │   ├── classification_feature_importance.png
│   │   └── classification_knn_sensitivity.png
│   └── module5_clustering/
│       ├── clustering_optimization_curves.png
│       ├── clustering_dendrogram_tree.png
│       └── clustering_master_topographies.png
│
└── results/                           # Summary CSV reports
    ├── module3_pattern_mining/
    │   └── academic_association_rules.csv
    ├── module4_classification/
    │   └── classification_metrics_summary.csv
    └── module5_clustering/
        └── clustering_metrics_summary.csv
```

## Run Order

```
1. python data_generator.py          → outputs/data/student_learning_behavior.csv
2. python module2_preprocessing.py   → outputs/data/preprocessed_*.csv + outputs/plots/module2_preprocessing/
3. python module3_pattern_mining.py  → outputs/results/module3_pattern_mining/
4. python module4_classification.py  → outputs/plots/module4_classification/ + outputs/results/module4_classification/
5. python module5_clustering.py      → outputs/plots/module5_clustering/ + outputs/results/module5_clustering/
```
